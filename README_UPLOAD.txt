# VERA v0.6.18 — GitHub Pages PWA pack

Upload **all files in this folder** to the root of the `elmtalk/VERA` GitHub repository.

The phone icon depends on `manifest.webmanifest`, `sw.js`, and the PNG icon files. Uploading only `index.html` will not provide the full installable PWA experience.

After GitHub Pages deploys, remove any previous VERA shortcut from the phone, open the site again, and choose **Add to Home screen** or **Install app** so the refreshed manifest and icon are used.
