Method note
FCIN12 is used to estimate indicative tree age from girth and growth context. Where a species is listed in FCIN12 Table 1a but the selected growth context is not tabulated, AVATA derives a context-adjusted estimate anchored to the nearest FCIN12 entry using context multipliers derived from FCIN12 calibration species. These derived values are explicitly flagged in the reasoning output. Species not listed in FCIN12 Table 1a require a proxy species.
Outputs are indicative diagnostic estimates derived from published arboricultural guidance. Results should be interpreted and applied only by a competent arboricultural professional.
