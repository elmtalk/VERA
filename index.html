How proxy selection works (FCIN12)
FCIN12 (White, 1998) requires a proxy species when the selected tree species is not listed in Table 1a. VERA will apply the selected proxy species’ Table 1a values together with the chosen growth context. Open British Arboricultural Record Dataset (OpenBARD) and the Veteran Tree Evidence Matrix (VTEM) continue to use the actual selected species/genus where available.

VERA reports FCIN12 and ATF separately. It does not blend FCIN12, ATF, the Open British Arboricultural Record Dataset (OpenBARD), the Veteran Architecture Index (VAI) or the Veteran Tree Evidence Matrix (VTEM) into a hidden score. The Veteran Tree Evidence Matrix (VTEM) stacks VERA-created evidence for rarity, replacement/continuity, habitat specialisation, count and catalogue size class where a real catalogue class exists; it does not multiply them. Where the catalogue gives only a minimum threshold, that threshold is used as a registration gate only, not as an invented size score. Where the selected TreM already encodes the catalogue size class, no second scale judgement is used. Ancient and borderline ancient evidence is identified first; otherwise the tree is assessed through age, size or condition pathways. Heritage and cultural notes are reported separately and do not alter structured significance values. Saved test trees are stored locally in this browser and can be exported as a single CSV with coordinates.

Method note
FCIN12 is used to estimate indicative tree age from girth and growth context. Where a species is listed in FCIN12 Table 1a but the selected growth context is not tabulated, VERA derives a context-adjusted estimate anchored to the nearest FCIN12 entry using context multipliers derived from FCIN12 calibration species. These derived values are explicitly flagged in the reasoning output. Species not listed in FCIN12 Table 1a require a proxy species.
Outputs are indicative diagnostic estimates derived from published arboricultural guidance. Results should be interpreted and applied only by a competent arboricultural professional.
